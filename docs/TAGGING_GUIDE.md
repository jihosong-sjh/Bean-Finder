# 커피 원두 태깅 가이드

문서 버전: v0.1  
작성일: 2026-05-08  
제품명: Bean Finder, 가칭  
연관 문서: [DATA_SPEC.md](./DATA_SPEC.md), [DATA_ENTRY_GUIDE.md](./DATA_ENTRY_GUIDE.md), [FUNCTIONAL_SPEC.md](./FUNCTIONAL_SPEC.md)  
문서 목적: 원두의 산미, 단맛, 쓴맛, 바디감, 향미, 컵노트, 추천 추출 방식을 일관되게 태깅하기 위한 기준을 정의한다.

## 1. 태깅 원칙

태깅은 사용자가 원두를 쉽게 찾고 비교하기 위한 운영자 판단값이다.

기본 원칙:

- 공식 로스터리 설명을 우선한다.
- 없는 정보를 과도하게 추정하지 않는다.
- 점수는 원두 간 상대 비교가 가능하도록 일관되게 준다.
- 입문자가 이해할 쉬운 표현으로 매핑될 수 있어야 한다.
- 판단이 어려우면 `3 보통`을 기본값으로 두되 근거를 남긴다.

## 2. 점수 공통 기준

맛 점수는 1-5 정수로 입력한다.

| 점수 | 의미 | 설명 |
| --- | --- | --- |
| 1 | 매우 낮음 | 거의 느껴지지 않음 |
| 2 | 낮음 | 약하게 느껴짐 |
| 3 | 보통 | 균형적이거나 중간 수준 |
| 4 | 높음 | 뚜렷하게 느껴짐 |
| 5 | 매우 높음 | 강하게 두드러짐 |

주의:

- 1과 5는 명확한 근거가 있을 때만 사용한다.
- 모든 항목을 3으로 입력하지 않는다.
- 단맛과 쓴맛은 서로 반대 개념이 아니다. 둘 다 높을 수 있다.
- 산미가 높다고 무조건 품질이 좋은 것은 아니다.

## 3. 산미 Acidity

산미는 밝고 상큼한 맛의 정도를 의미한다.

| 점수 | 기준 | 예시 표현 |
| --- | --- | --- |
| 1 | 산미가 거의 없음 | 신맛 적음, 묵직함, 다크 |
| 2 | 산미가 낮음 | 부드러움, 고소함 |
| 3 | 중간 산미 | 균형감, 은은한 산미 |
| 4 | 산미가 뚜렷함 | 상큼함, 과일향 |
| 5 | 산미가 매우 강함 | 레몬, 자몽, 와인 같은 산미 |

점수 판단 힌트:

- 라이트 로스팅은 산미가 높을 가능성이 있다.
- 에티오피아, 케냐 계열은 산미가 높을 가능성이 있다.
- 브라질, 과테말라, 인도네시아 계열은 산미가 낮거나 중간일 가능성이 있다.
- 다크 로스팅은 산미가 낮게 느껴질 가능성이 있다.

쉬운 표현 매핑:

| 조건 | tag |
| --- | --- |
| acidity 1-2 | 신맛 적음 |
| acidity 4-5 | 산미 있음 |

## 4. 단맛 Sweetness

단맛은 설탕 같은 직접적 단맛이 아니라 커피에서 느껴지는 단맛의 인상이다.

| 점수 | 기준 | 예시 표현 |
| --- | --- | --- |
| 1 | 단맛이 거의 없음 | 건조함, 쌉쌀함 |
| 2 | 단맛이 약함 | 담백함 |
| 3 | 보통 | 균형감 |
| 4 | 단맛이 뚜렷함 | 캐러멜, 꿀, 과일 단맛 |
| 5 | 단맛이 매우 강함 | 농축된 과일, 시럽 같은 단맛 |

점수 판단 힌트:

- 캐러멜, 꿀, 브라운슈가, 초콜릿 표현은 단맛 점수를 높일 수 있다.
- 내추럴 가공은 단맛이 높게 느껴질 가능성이 있다.
- 단맛이 높아도 산미가 함께 높을 수 있다.

쉬운 표현 매핑:

| 조건 | tag |
| --- | --- |
| sweetness 4-5 | 단맛 좋음 |

## 5. 쓴맛 Bitterness

쓴맛은 다크 초콜릿, 카카오, 로스티드, 탄맛 계열의 쌉쌀함을 의미한다.

| 점수 | 기준 | 예시 표현 |
| --- | --- | --- |
| 1 | 쓴맛이 거의 없음 | 깨끗함, 밝음 |
| 2 | 쓴맛이 낮음 | 부드러움 |
| 3 | 중간 쓴맛 | 균형 잡힌 쌉쌀함 |
| 4 | 쓴맛이 뚜렷함 | 다크초콜릿, 카카오 |
| 5 | 쓴맛이 매우 강함 | 탄맛, 스모키, 강한 다크 |

점수 판단 힌트:

- 다크 로스팅은 쓴맛이 높을 가능성이 있다.
- 초콜릿 표현은 쓴맛이 아니라 단맛/바디와 함께 판단한다.
- 탄맛, 스모키, 로스티드 표현은 쓴맛 점수를 높인다.

쉬운 표현 매핑:

| 조건 | tag |
| --- | --- |
| bitterness 1-2 | 쓴맛 적음 |

## 6. 바디감 Body

바디감은 커피의 무게감, 질감, 입안에서 느껴지는 밀도를 의미한다.

| 점수 | 기준 | 예시 표현 |
| --- | --- | --- |
| 1 | 매우 가벼움 | 차 같은 질감, 산뜻함 |
| 2 | 가벼움 | 깔끔함, 밝음 |
| 3 | 보통 | 균형감 |
| 4 | 묵직함 | 크리미함, 밀도감 |
| 5 | 매우 묵직함 | 진함, 두꺼움, 강한 무게감 |

점수 판단 힌트:

- 라이트 로스팅과 워시드 계열은 바디가 낮을 가능성이 있다.
- 미디엄다크, 다크 로스팅은 바디가 높을 가능성이 있다.
- 라떼용, 에스프레소용 설명은 바디가 높을 가능성이 있다.
- 초콜릿, 견과류, 캐러멜 표현은 바디 점수를 높일 수 있다.

쉬운 표현 매핑:

| 조건 | tag |
| --- | --- |
| body 1-2 | 가벼움 |
| body 4-5 | 묵직함 |

## 7. 향미 강도 Aroma

향미 강도는 향의 선명함과 복합성을 의미한다.

| 점수 | 기준 |
| --- | --- |
| 1 | 향 표현이 거의 없음 |
| 2 | 향이 약함 |
| 3 | 기본적인 향이 있음 |
| 4 | 향이 뚜렷함 |
| 5 | 향이 매우 선명하고 복합적임 |

입력 기준:

- 공식 컵노트가 구체적이고 다양한 경우 4-5를 고려한다.
- 컵노트가 단순하거나 정보가 부족하면 3 이하를 고려한다.
- 판단이 어려우면 `null`로 둘 수 있다.

## 8. 밸런스 Balance

밸런스는 산미, 단맛, 쓴맛, 바디감이 조화롭게 느껴지는 정도다.

| 점수 | 기준 |
| --- | --- |
| 1 | 특정 맛이 지나치게 튐 |
| 2 | 균형이 약함 |
| 3 | 무난한 균형 |
| 4 | 조화로움 |
| 5 | 매우 완성도 높은 균형 |

입력 기준:

- 입문자 추천 원두는 보통 balance 3-5 범위가 적합하다.
- 호불호가 강한 산미/발효향 원두는 balance를 낮게 줄 수 있다.
- 판단이 어려우면 `null`로 둘 수 있다.

## 9. 로스팅 태깅

`roast_level` 허용 값:

| 값 | 표시명 | 기준 |
| --- | --- | --- |
| light | 라이트 | 산미와 향미가 강조됨 |
| medium | 미디엄 | 산미, 단맛, 바디의 균형 |
| medium_dark | 미디엄다크 | 단맛, 바디감, 고소함 강조 |
| dark | 다크 | 쓴맛, 묵직함, 로스티드 향 강조 |

판단 기준:

- 공식 로스팅 표기를 우선한다.
- `약배전`은 light로 매핑한다.
- `중배전`은 medium으로 매핑한다.
- `중강배전`은 medium_dark로 매핑한다.
- `강배전`은 dark로 매핑한다.

## 10. 컵노트 그룹

컵노트는 표준 key와 group으로 관리한다.

| group | 설명 | 예시 key |
| --- | --- | --- |
| fruit | 과일 | citrus, apple, peach |
| berry | 베리 | berry, blueberry, strawberry |
| floral | 꽃 | floral, jasmine |
| nut | 견과 | nutty, almond, hazelnut |
| chocolate | 초콜릿 | chocolate, cacao |
| sweet | 단맛 | caramel, honey, brown_sugar |
| spice | 향신료 | cinnamon, clove |
| herbal | 허브 | tea, herb |
| roasted | 로스티드 | smoky, roasted_nut |
| other | 기타 | winey |

## 11. 컵노트 표준화 기준

공식 표현을 그대로 저장하지 않고 표준 key로 매핑한다.

| 원문 표현 | 표준 key | group |
| --- | --- | --- |
| 레몬, 오렌지, 자몽 | citrus | fruit |
| 블루베리, 딸기, 라즈베리 | berry | berry |
| 재스민, 꽃향 | floral 또는 jasmine | floral |
| 아몬드, 헤이즐넛 | almond 또는 hazelnut | nut |
| 견과류, 너티 | nutty | nut |
| 초콜릿, 다크초콜릿 | chocolate | chocolate |
| 카카오 | cacao | chocolate |
| 캐러멜 | caramel | sweet |
| 꿀 | honey | sweet |
| 홍차, 차 | tea | herbal |
| 와인 같은 | winey | other |

규칙:

- 구체 key가 있으면 구체 key를 사용한다.
- 구체 key가 없으면 상위 key를 사용한다.
- 표준 사전에 없는 표현은 임시로 추가하지 않고 운영자 검토 목록에 둔다.

## 12. 쉬운 맛 표현 태그

`easy_taste_tags`는 운영자가 직접 입력하지 않고 규칙으로 생성한다.

| 조건 | tag |
| --- | --- |
| acidity 1-2 | 신맛 적음 |
| acidity 4-5 | 산미 있음 |
| body 1-2 | 가벼움 |
| body 4-5 | 묵직함 |
| bitterness 1-2 | 쓴맛 적음 |
| sweetness 4-5 | 단맛 좋음 |
| roast_level = dark | 진하고 쌉쌀함 |
| tasting_notes group = nut | 고소함 |
| tasting_notes group = chocolate | 초콜릿 느낌 |
| tasting_notes group = berry | 베리향 |
| tasting_notes group = floral | 꽃향 |
| recommended_brew_methods에 latte 포함 | 라떼용 |
| recommended_brew_methods에 hand_drip 포함 | 핸드드립용 |

## 13. 추천 추출 방식 태깅

허용 값:

- hand_drip
- espresso
- latte
- cold_brew
- moka_pot
- french_press

### 13.1 hand_drip

추천 조건:

- 산미와 향미가 뚜렷함
- 라이트 또는 미디엄 로스팅
- 플로럴, 베리, 시트러스 계열 컵노트
- 깔끔하고 가벼운 바디

### 13.2 espresso

추천 조건:

- 단맛과 바디감이 있음
- 미디엄, 미디엄다크, 다크 로스팅
- 초콜릿, 견과류, 캐러멜 계열
- 산미가 너무 날카롭지 않음

### 13.3 latte

추천 조건:

- 바디감 4-5
- 산미 1-3
- 쓴맛 또는 초콜릿 계열이 어느 정도 있음
- 우유와 섞였을 때 맛이 묻히지 않을 원두

### 13.4 cold_brew

추천 조건:

- 쓴맛이 낮거나 중간
- 단맛이 있음
- 과일향, 초콜릿, 고소함 계열 모두 가능
- 산미가 부드럽게 느껴질 가능성이 있는 원두

## 14. 카테고리 태깅 기준

카테고리는 직접 입력하지 않고 조건으로 계산한다.

| 카테고리 | 조건 |
| --- | --- |
| 신맛 적은 원두 | acidity 1-2 |
| 산미 있는 원두 | acidity 4-5 |
| 고소한 원두 | tasting_notes group nut 또는 고소함 tag |
| 묵직한 원두 | body 4-5 |
| 라떼용 원두 | brew_method latte 또는 espresso |
| 핸드드립용 원두 | brew_method hand_drip |
| 2만원 이하 원두 | price <= 20000 |
| 가성비 원두 | price_per_100g 하위 30% |
| 디카페인 원두 | is_decaf true |

## 15. 입문자 추천 태깅 기준

입문자 추천 랭킹 조건:

- acidity 1-3
- bitterness 1-3
- price <= 25000
- balance가 있으면 3 이상
- 지나치게 발효향이 강한 컵노트는 제외 또는 후순위

권장 특성:

- 맛 설명이 이해하기 쉬움
- 산미나 쓴맛이 과하지 않음
- 가격 부담이 낮음
- 핸드드립 또는 아메리카노로 무난함

## 16. 라떼 추천 태깅 기준

라떼 추천 원두의 권장 특성:

- body 4-5
- acidity 1-3
- sweetness 3-5
- bitterness 2-4
- roast_level medium_dark 또는 dark 우선
- chocolate, nut, caramel, roasted 계열 컵노트

주의:

- 산미가 5인 원두는 라떼 추천에서 후순위로 둔다.
- 바디가 1-2인 원두는 라떼에서 맛이 약할 수 있다.

## 17. 핸드드립 추천 태깅 기준

핸드드립 추천 원두의 권장 특성:

- aroma 3-5
- acidity 2-5
- bitterness 1-3
- roast_level light 또는 medium 우선
- fruit, berry, floral, herbal 계열 컵노트

주의:

- 다크 로스팅도 핸드드립 가능하지만 MVP 추천에서는 후순위로 둔다.
- 산미가 낮아도 향미가 좋으면 hand_drip을 포함할 수 있다.

## 18. 태깅 예시

### 18.1 산미 있는 라이트 로스팅

공식 설명:

```text
에티오피아 내추럴, 라이트 로스팅, 베리, 꽃향, 시트러스
```

태깅:

```json
{
  "roast_level": "light",
  "tasting_notes": ["berry", "floral", "citrus"],
  "taste_profile": {
    "acidity": 4,
    "sweetness": 4,
    "bitterness": 1,
    "body": 2,
    "aroma": 5,
    "balance": 4
  },
  "recommended_brew_methods": ["hand_drip", "cold_brew"]
}
```

### 18.2 라떼용 묵직한 원두

공식 설명:

```text
브라질 블렌드, 미디엄다크, 초콜릿, 견과류, 묵직한 바디
```

태깅:

```json
{
  "roast_level": "medium_dark",
  "tasting_notes": ["chocolate", "nutty"],
  "taste_profile": {
    "acidity": 2,
    "sweetness": 4,
    "bitterness": 3,
    "body": 4,
    "aroma": 3,
    "balance": 4
  },
  "recommended_brew_methods": ["espresso", "latte"]
}
```

### 18.3 디카페인 원두

공식 설명:

```text
콜롬비아 디카페인, 미디엄 로스팅, 캐러멜, 견과류, 부드러운 산미
```

태깅:

```json
{
  "roast_level": "medium",
  "tasting_notes": ["caramel", "nutty"],
  "taste_profile": {
    "acidity": 2,
    "sweetness": 3,
    "bitterness": 2,
    "body": 3,
    "aroma": 3,
    "balance": 4
  },
  "recommended_brew_methods": ["hand_drip", "espresso"],
  "flags": {
    "is_decaf": true
  }
}
```

## 19. 태깅 검수 체크리스트

- 산미, 단맛, 쓴맛, 바디감이 모두 1-5 정수다.
- 컵노트가 표준 key로 입력되어 있다.
- 공식 설명과 점수가 크게 충돌하지 않는다.
- 로스팅과 맛 점수가 대체로 일관된다.
- 추천 추출 방식이 1개 이상 있다.
- 라떼용 원두는 바디감과 로스팅을 확인했다.
- 핸드드립용 원두는 향미와 쓴맛을 확인했다.
- 입문자 추천 가능 원두는 산미와 쓴맛이 과하지 않다.
- 판단이 어려운 선택 점수는 `null`로 둘 수 있다.

## 20. 보정 기준

출시 후 사용자 피드백이나 리뷰 기반으로 점수를 보정할 수 있다.

보정이 필요한 경우:

- 사용자가 "신맛이 강하다"고 반복 피드백
- 사용자가 "라떼에 약하다"고 반복 피드백
- 판매처 설명이 변경됨
- 로스팅 포인트가 변경됨
- 같은 원두의 최신 배치 특성이 크게 달라짐

보정 원칙:

- 한두 개 의견만으로 점수를 바꾸지 않는다.
- 공식 정보 변경은 즉시 반영한다.
- 사용자 피드백은 여러 건이 쌓이면 반영한다.
- 변경 시 `updated_at`과 근거를 남긴다.
