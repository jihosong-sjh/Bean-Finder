## 1. 사용 방법

이 문서는 구현 중 계속 갱신한다.

체크 규칙:

- 작업이 완료되면 `[ ]`를 `[x]`로 바꾼다.
- 진행 중인 작업은 `상태`에 `진행 중`으로 표시한다.
- 막힌 작업은 `상태`에 `블로커`로 표시하고 `블로커 로그`에 기록한다.
- 완료된 마일스톤은 `마일스톤 현황`에서 체크한다.
- 테스트나 QA 없이 완료 처리하지 않는다.

상태 값:

| 상태 | 의미 |
| --- | --- |
| 대기 | 아직 시작하지 않음 |
| 진행 중 | 작업 중 |
| 검토 중 | 구현 완료, 리뷰 또는 QA 대기 |
| 완료 | 구현과 검증 완료 |
| 블로커 | 외부 결정이나 문제로 진행 불가 |
